## Основные команды для работы c Git 

1. git init — инициализация локального репозитория

2. git status — получить информацию OT git о его текущем состоянии
3. git add — добавить файл или файлы к следующему коммиту

4. git commit -m “message” — создание коммита.

5. git log — вывод на экран истории всех коммитов с их хеш-кодами
6. git branch — посмотреть список веток в репозитории

7. git branch <название ветки> — создать новую ветку

8. git checkout <название ветки> — переход к другой ветке

9. git branch -d <название ветки> — удалить ветку

10. git checkout -b "Название ветки" - Создает и переходит в данную ветку с помощью только 1 команды &

### Команды для работы с удаленным репозиторием 

11. git clone <url-agpec репозитория> — клонирование внешнего репозитория на
локальный ПК
12. git pull — получение изменений и слияние с локальной версией

13. git push — отправляет локальную версию репозитория на внешний
